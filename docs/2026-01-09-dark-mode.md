# Dark Mode Toggle

**Date:** 2026-01-09

## Overview

Add a dark mode toggle to the application that allows users to switch between light and dark themes.

## Requirements

- Add a toggle control to switch between light and dark modes
- Persist the user's theme preference
- Support system preference detection (prefers-color-scheme)
- Ensure smooth transition between themes

## Acceptance Criteria

- [ ] Dark mode toggle is visible and accessible in the UI
- [ ] Theme preference persists across sessions
- [ ] All UI components properly support both themes
- [ ] Respects system dark mode preference by default
