# Task Model Update

## Task Properties
- **hasDeadline**: boolean - whether task has a deadline
- **canDoAnytime**: boolean - whether task can be done before deadline

## Tabs
| Tab | Contains |
|-----|----------|
| **Focus** | Tasks where deadline has been reached |
| **Might Do** | Tasks marked as `canDoAnytime: true` (no deadline constraint) |
| **Projects** | Projects with linked subtasks |
| **Done** | Completed tasks |
| **Inbox** | Tasks with `hasDeadline: false` AND `canDoAnytime: false` (undefined when/how) |

## Task Entry
When adding tasks, clarify:
1. Does it have a deadline?
2. Can it be done anytime (before deadline)?
